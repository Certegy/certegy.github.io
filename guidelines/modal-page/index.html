Modal Page

Note

Please note that this documentation is still a work in process

When a user clicks on the banner rather than link away from your website we recommend having a modal to appear that explains the benifits of our service. In this page we will explain two methods to achieving this however these are merely guides. If you have a pre installed modal that difffers from the one we have we recommend that you use that one instead.

Option 1:

Our modal of choice is called JQuery Model and this option will be demonstrated using it. To install JQuery Model please follow the instruction laid out in here JQuery Modal Installation Guide. Once you have completed installation please follow the seps below.

Step one is to go to here and follow the steps to install this.

Option 2:

Bootstrap Modal, if you are using bootstrap then we recomend that you use this option.
